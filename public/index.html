This API lets you generate images using Fal.ai FLUX or similar models.

How to use:

Send a POST request to:
/api/generate

with JSON body:

{ "prompt": "a cyberpunk city at night", "width": 1024, "height": 1024 }

Example:

curl -X POST https://txttoimg.vercel.app/api/generate \ -H "Content-Type: application/json" \ -d '{"prompt":"a cat wearing sunglasses"}'

The API will return a JSON response containing an image URL or error info.
Example: { "ok": true, "url": "https://..." }
